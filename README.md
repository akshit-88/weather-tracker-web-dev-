Weather Tracker 🌦️
Overview

The Weather Tracking Web Application is a web-based project developed as part of a Web Development assignment. The application allows users to check the current weather conditions of different cities.

Users can enter the name of a city, and the system retrieves weather information such as temperature, humidity, and weather conditions. The application uses HTML for the frontend interface and Java for handling the backend logic and API communication.

Features

Search weather by city name

Display temperature

Show humidity

Show wind speed

Display weather conditions

Simple and user-friendly interface

Technologies Used

HTML5 – Used for structuring the web interface

CSS3 – Used for styling the webpage

Java – Used for backend logic and fetching weather data

Weather API – Used to retrieve real-time weather data

Project Structure
weather-tracker/

How It Works

The user enters the city name in the input field.

The request is processed by the Java program.

The Java program sends a request to a Weather API.

The API returns weather data in JSON format.

The application extracts key details such as:

Temperature

Humidity

Weather condition

The information is displayed to the user on the webpage.

How to Run the Project

Download or clone the project.

Open the project folder.

Run the Java backend file.

Open index.html in a browser to access the application.
